# NLP_2025W - Spoiler Identification
deliverable for course NLP given at WUT.

Repository contains source codes for comparison of different methodologies for Spoiler Identifications

### Structure of Repository:

- EDA and Merge notebooks contain preprocessing, analysis and structuring raw datasets
- data directory is placeholder directory for datasets (see dedicated README)
- Training directory contains scripts and notebooks used to train ML models
- Models directory contains serialized models
- Tester directory contains scripts and notebooks used to tested previously trained models
