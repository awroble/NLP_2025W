# Narrative-Aware Topic Discovery for News Articles

This repository contains code and the final report for the project
"Topic Discovery for News Articles with Narrative Analysis".

## Structure
- `dataset_creation/`: notebooks and scripts used for dataset creation
- `clustering/`: notebooks and scripts used for experiments
- pdf with final report

## Dataset avaliability
url: https://huggingface.co/datasets/Tytanito/news-narratives-embeddings

## Reproducibility
Experiments were run using fixed random seeds. Some GPU-based operations
may introduce minor non-determinism.
random_state=42
