Directory contains files about cultural section of the project.

Datasets with prompts are in nlp(cultural).csv and nlp(cultural_pictures).csv. The llm test process is possible via culture_test.ipynb and culture_test_pics.ipynb. 

Results are avaible in results directory. results.csv contains results, final prompts received from llms are in result_prompts.csv.
Results from multimodal prompts are in qwen_prompts_results
